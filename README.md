# WORDLE Clone

The clone project of [WORDLE](https://www.nytimes.com/games/wordle/index.html), the popular web-based word-guessing game. Built with React, Typescript and Redux

## Tech Stack

### Frontend ( Only )

- React
- Typescript
- Redux
